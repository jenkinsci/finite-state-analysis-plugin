Selects which Finite State platform (and transport) this build step targets:
  • Alloy (legacy CLT) — the default. Downloads and runs the Finite State command-line tool (CLT) on the build agent, exactly as previous plugin versions did. Requires Java on the agent. Choose this if your organization is still on the Alloy platform.
  • Helix (public v0 API) — calls the Finite State public v0 REST API directly. No CLT download and no Java required on the agent. Choose this only if your organization has been migrated to the Helix platform.
Jobs saved before this option existed default to Alloy, so upgrading the plugin does not change their behavior.
